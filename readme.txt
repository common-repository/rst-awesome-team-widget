== Description ==

RST Awesome Team Widget is an elementor add-on to showcase your Team widget style/Designe
This plugin is responsive and has 10 layouts inside the plugin.This plugin can be easily used by a user by drag and drop.

Each section of this plugin can be easily changed by a user 

This Team Widget Plugin is 100% customizable easily. You can summon the Team widget onto any page on which you need to showcase RST Awesome Team Widget or grid view. It offers 10 variations of the Team layouts.

You also get to customize the Widget' fields, style, color, and general appearance. With this WordPress RST Awesome Team Widget plugin, you can control whether the Team images on the Widget you display are square or rounded. The RST Awesome Team Widget.

We will update this plugin in the future so that a user can get everything he needs from this plugin.
